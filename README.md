🚀 AI Marketing Generator

An AI-powered application that automatically generates marketing content including images, videos, captions, and voiceovers from a simple product description.

🧠 Overview

AI Marketing Generator allows users to create ready-to-use advertisements by simply providing:

Product name
Description
Desired style (e.g., modern, luxury, funny)

The system uses AI to generate:

📝 Captions & hashtags
🖼️ Marketing images
🔊 Voiceovers (Text-to-Speech)
🎬 Promotional videos

Frontend

Backend
